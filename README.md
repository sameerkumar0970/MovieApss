# MovieApss

MovieApss is a simple movie listing and cart web app built using HTML, CSS, and JavaScript. It allows users to browse movies, view details, and add movies to their cart.

## Features

- Movie catalog listing
- Single movie detail page
- Add to cart functionality
- Simple, clean UI with CSS styling
- Data stored in JSON file
- Responsive design for mobile and desktop

## Files Structure

- `index.html`, `index.js`, `index.css`: Main homepage and script
- `single.html`: Movie detail page
- `cart.html`, `cart.js`, `cart.css`: Cart page and its logic
- `db.json`: Movie data
- `package.json`, `package-lock.json`: Dependencies (if using npm or any build tools)

## How to Run

1. Clone this repository.
2. Open `index.html` in your browser.
3. Browse movies and use cart feature.
